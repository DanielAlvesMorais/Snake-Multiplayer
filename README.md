## 🎮 Sobre o Jogo
Uma versão para dois jogadores do clássico jogo da cobrinha. [cite_start]Ambos os jogadores compartilham a mesma tela e competem para comer a maior quantidade de comida sem colidir[cite: 108]. [cite_start]O jogo roda em tempo real e utiliza o framework LibGDX para gráficos e captura de comandos[cite: 109].
